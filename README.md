# Privacy Policy for NotebookLM Quick Add

Last updated: December 31, 2024

## Overview
NotebookLM Quick Add is a Chrome extension that helps users add sources to NotebookLM quickly.

## Data Collection
This extension does NOT collect, transmit, or share any personal data.

## Local Storage
The extension stores the following data locally on your device using Chrome's storage API:
- History of recently added sources (up to 50 items)

This data never leaves your device and is not accessible to the developer or any third party.

## Permissions
The extension requires certain permissions solely to provide its core functionality:
- Access to notebooklm.google.com to automate the add source workflow
- Storage permission to save history locally

## Third-Party Services
This extension does not use any third-party analytics or tracking services.

## Contact
If you have questions about this privacy policy, please open an issue on GitHub.

Any changes to this policy will be reflected on this page.
